# 📧 Email Checker

أداة بسيطه مكتوبة بـ Python للكشف عن تسريبات البيانات المرتبطة بأي إيميل.

---

## ✨ المميزات

- فحص الإيميل من خلال مصدرين مختلفين: **XposedOrNot** و **HackCheck**
- عرض تفاصيل كاملة عن كل تسريب (الموقع، التاريخ، عدد السجلات، نوع البيانات المسربة)
- تقييم مستوى خطورة كلمة المرور
- حفظ تقرير كامل تلقائياً بعد كل فحص
- دعم فحص إيميل واحد أو قائمة إيميلات من ملف txt

---

## ⚙️ المتطلبات

- Python 3.6 أو أحدث
ما تحتاج أي مكتبات خارجية

---

## 🚀 طريقة الاستخدام

**فحص إيميل واحد:**
```bash
python email_checker.py example@gmail.com
```

**فحص قائمة إيميلات من ملف:**
```bash
python email_checker.py emails.txt
```

---

## 📄 التقرير

بعد كل فحص يتم حفظ تقرير txt تلقائي بالاسم:
```
report_example@gmail.com_20260222_120000.txt
```
